# Student Academic Data Analyzer

A Python-based data analysis project that evaluates student academic performance using statistical methods and visualization.

This project demonstrates how Python can be used for analyzing datasets, calculating statistical metrics, and generating visual insights.

---

## Features

- Reads student dataset from a CSV file
- Calculates subject-wise averages
- Computes variance and standard deviation
- Performs gender-wise performance analysis
- Visualizes results using bar charts

---

## Technologies Used

- Python
- Pandas
- Matplotlib

## How to Run the Project

1. Clone the repository

git clone https://github.com/YOURUSERNAME/student-performance-data-analysis.git

2. Navigate to the project folder

cd student-performance-data-analysis

3. Install required libraries

pip install pandas matplotlib

4. Run the program

python analyzer.py
